# Submission Checklist and Email

## Repository checks

- [ ] The GitHub repository opens in a logged-out browser window.
- [ ] The default branch is `main` and the README renders correctly.
- [x] No secrets, IDE files, build outputs, or local databases are tracked.
- [x] `./gradlew test ktlintCheck` passes from a fresh clone.
- [x] Swagger UI, complete results, partial timeout results, and audit persistence have been demonstrated.

## Video checks

- [ ] Length is between 5 and 10 minutes.
- [ ] Audio, editor text, terminal output, and diagrams are readable.
- [ ] The video covers design, implementation, AI collaboration, success, partial failure, tests, and production tradeoffs.
- [ ] The video link opens in a private/incognito browser window without requesting access.
